Transform project ini menjadi SaaS "Undangan Digital Pernikahan" fullstack yang production-ready, dengan Admin Dashboard super lengkap untuk mengatur seluruh website, landing page, template, paket harga, pesanan, dan data pelanggan.

VISI PRODUK
Project ini harus berubah dari landing page statis menjadi platform bisnis undangan digital pernikahan yang bisa dijual ke banyak pelanggan.
Sistem harus mendukung:
- landing page marketing
- admin panel utama
- customer dashboard
- manajemen template
- manajemen paket harga
- pembuatan undangan
- order management
- pembayaran manual / status pembayaran
- RSVP & ucapan tamu
- analytics sederhana
- SEO-ready
- mobile responsive
- siap deploy

TECH STACK
Tetap gunakan stack project yang sudah ada:
- React + Vite
- Express.js
- PostgreSQL
- Drizzle ORM
- Tailwind CSS
- shadcn/ui jika perlu
- session-based auth atau JWT yang aman
- upload image support
- environment variables rapi
- struktur folder modular dan scalable
