Upgrade fitur Guest Management dan RSVP pada project Wed Invitation Hub agar menjadi sistem RSVP profesional untuk undangan digital pernikahan.

Jangan rewrite project dari nol.
Jangan merusak fitur lain seperti auth, template builder, payment, invitations, admin dashboard, pricing, billing, dan halaman publik.

Fokus hanya pada:
- guest management
- RSVP
- import tamu
- link undangan per tamu
- WhatsApp share
- QR check-in

==================================================
TUJUAN
==================================================

Pemilik undangan harus bisa:

1. tambah tamu manual
2. import tamu dari Excel / CSV
3. generate link undangan unik untuk tiap tamu
4. nama tamu otomatis muncul di link undangan
5. kirim link undangan ke WhatsApp
6. lihat RSVP per tamu
7. export RSVP
8. scan/check-in tamu saat acara dengan QR code

Tamu undangan harus bisa:

9. membuka link undangan personal
10. melihat namanya di undangan
11. mengisi RSVP
12. mengirim ucapan
13. menerima QR code/check-in code jika diperlukan

==================================================
STEP 4 — IMPORT EXCEL / CSV
==================================================

Tambahkan fitur import file Excel / CSV.

Format file:
Name | Phone | Email | Max Guest

Gunakan library seperti:
- xlsx
- papaparse

Flow:
1. user upload file
2. parse file
3. validasi data
4. tampilkan preview sebelum import jika memungkinkan
5. simpan ke database guests
6. generate guest_code dan qr_code_value otomatis

Validasi:
- name wajib
- invitation_id valid
- max_guest angka valid
- duplikasi tamu dicegah atau ditangani dengan aman

Tambahkan tombol:
- Download Template Excel

==================================================
STEP 5 — GENERATE LINK UNDANGAN PER TAMU
==================================================

Setiap tamu harus punya link undangan unik.

Contoh format:
- /invite/:slug?to=Ahmad
atau lebih rapi:
- /invite/:slug?guest=GUESTCODE123

Lebih baik jika mendukung keduanya:
- nama tamu untuk tampilan
- guest_code untuk identifikasi sistem

Saat tamu membuka link:
- nama tamu muncul di cover undangan
- guest terkait bisa dikenali oleh sistem
- RSVP bisa otomatis dikaitkan ke guest_id

==================================================
STEP 6 — WHATSAPP SHARE
==================================================

Tambahkan tombol share WhatsApp untuk tiap tamu.

Format pesan WhatsApp:
- sapaan tamu
- nama pasangan
- link undangan personal
- ajakan untuk mengisi RSVP

Contoh:
Halo Bapak/Ibu {{guest_name}}, kami mengundang Anda ke acara pernikahan kami. Berikut link undangannya: {{guest_link}}

Tombol:
- Kirim WhatsApp
- Copy Link

Gunakan URL WhatsApp yang valid agar langsung terbuka.

STEP 10 — QR CODE CHECK-IN
==================================================

Tambahkan sistem QR check-in tamu.

Untuk setiap guest:
- generate qr_code_value unik
- generate QR code image atau data URL jika perlu

Tambahkan halaman check-in:
- /checkin
atau
- /admin/checkin

Flow:
1. admin/staff scan QR code tamu
2. sistem cari guest berdasarkan qr_code_value
3. tandai tamu sebagai checked_in
4. simpan checked_in_at

Tambahkan juga fallback check-in manual:
- cari tamu berdasarkan nama / phone / guest_code
- klik tombol Check-in

Validasi:
- tamu yang sudah checked-in tidak boleh dicatat dua kali tanpa peringatan
- tampilkan status jika QR code tidak valid
