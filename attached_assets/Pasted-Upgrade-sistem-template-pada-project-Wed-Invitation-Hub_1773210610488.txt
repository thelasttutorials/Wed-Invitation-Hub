Upgrade sistem template pada project Wed Invitation Hub menjadi visual builder drag & drop yang powerful seperti Canva / Framer / Webflow, tetapi khusus untuk membuat template undangan digital.

Jangan rewrite project dari nol.
Jangan merusak fitur yang sudah berjalan seperti auth, pricing, billing, RSVP, wishes, dan invitation.

Fokus pada peningkatan sistem builder template.

==================================================
TUJUAN
==================================================

Membuat Admin Template Builder visual yang memungkinkan admin membuat template undangan digital dengan drag & drop.

Admin harus bisa:

- drag section
- reorder section
- edit content
- edit style
- preview realtime
- save template
- publish template
- duplicate template

Template yang dibuat harus benar-benar bisa digunakan oleh invitation publik.

==================================================
BUILDER LAYOUT
==================================================

Buat layout builder seperti page builder modern.

Struktur UI:

Left Sidebar
- daftar section yang bisa ditambahkan

Center Canvas
- live preview template
- area drag & drop

Right Sidebar
- property editor
- style editor

Top Toolbar
- save
- publish
- preview desktop/mobile
- undo/redo jika memungkinkan
